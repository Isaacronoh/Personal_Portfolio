Project Overview

I am on process developing Wajir East Technical and Vocational College website. The purpose of it is to enhance its digital presence and improve service delivery. The system will support online student course applications, automated admission letters via email, and admin features to approve or reject applicants.

Key Features

  • Responsive landing page showcasing all school departments
  • Online course application form with database integration
  • Admin dashboard for approving applications
  • Automatic email system to send admission letters
  • Django-powered backend with dynamic content
